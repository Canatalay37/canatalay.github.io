HW2 - Internet Society Research

a) Why was the Internet Society created?

Answer: The Internet Society was created in 1992 by early Internet pioneers to support the open development, evolution, and use of the Internet for the benefit of people around the world. Its mission is centered on keeping the Internet open, globally connected, secure, and trustworthy. The organization also supports education, technical standards, public policy, and community collaboration related to Internet development. In other words, the Internet Society was established to help protect the future of the Internet and make sure it continues to serve society in a positive and accessible way.

b) Determine which local chapter is closest to you. Visit its website. List the website’s URL and an activity or service that the chapter provides.

Answer: Since I study at Near East University in Northern Cyprus, the closest Internet Society chapter is most likely the Turkey Chapter. This is a geographic inference based on proximity, since Northern Cyprus does not appear to have its own separate chapter listed in the sources I found. The Internet Society does list Turkey as an official chapter.

Chapter URL:
https://www.internetsociety.org/chapters/turkey/

Example activity or service: A strong example connected with the Turkey Chapter is support for projects that promote digital inclusion and Internet awareness. One Internet Society project in Turkey highlighted coding education for girls through the “Coding Sisters” initiative, showing how the chapter’s local ecosystem supports Internet education, digital skills, and community empowerment.

c) How can you join the Internet Society? What does it cost to join? Would you recommend that a beginning Web developer join the Internet Society? Why or why not?

Answer: A person can join the Internet Society by becoming an individual member through its official website. The organization’s membership page indicates that individuals can become members, and the current public membership path is presented as a free individual membership option.

Membership URL:
https://www.internetsociety.org/become-a-member/

Cost: Individual membership is free.

Recommendation: Yes, I would recommend that a beginning Web developer join the Internet Society. It offers useful learning resources, exposure to Internet standards, access to a global technical community, and a broader understanding of how the Internet works beyond only writing HTML and CSS. For a beginner, this is valuable because professional web development also involves awareness of security, accessibility, policy, and Internet infrastructure. Joining the Internet Society can therefore help a new developer become more informed, responsible, and professionally prepared. This recommendation is my own conclusion based on the organization’s mission, learning opportunities, and membership resources.

1. Missing Code Completion

Task: Fill in the missing code so that the background and text colors have good contrast, and the header area uses Arial.

Solution: Since the background color is a dark blue (#0066CC), the text color is set to white (#FFFFFF) for strong contrast and better readability. The header font is set to Arial. In addition, the missing HTML tags are completed correctly so the page structure becomes valid.

2. Find the Error

Task: Explain why the page corresponding to the given code does not display properly in a browser.

Solution: The error is located at the end of the CSS block. The code opens a second <style> tag instead of closing the first one with </style>. Because of this mistake, the browser cannot correctly separate the CSS from the HTML, which causes the page to display improperly.

<!DOCTYPE html> <html lang="en"> <head> <title>Trillium Media Design</title> <meta charset="utf-8"> <style> body { background-color: #000066; color: #CCCCCC; font-family: Arial, sans-serif; font-size: 1.2em; } </style> </head> <body> <header><h1>Trillium Media Design</h1></header> <main> <p>Our professional staff takes pride in its working relationship with our clients by offering personalized services that listen to their needs, develop their target areas, and incorporate these items into a website that works.</p> </main> </body> </html>
